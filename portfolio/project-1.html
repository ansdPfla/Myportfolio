AI 기반 얼굴인식 출석체크 시스템

인공지능 기술을 활용하여 개인의 얼굴 특징을 인식하고 자동으로 출석을
확인 및 기록하는 비접촉식 스마트 관리 시스템입니다.

📌 프로젝트 소개

이 프로젝트는 인공지능(AI)의 딥러닝 기술을 활용하여 자동화된 비접촉식 출석 관리 솔루션을 구축하는 것을 목표로 합니다.
기존의 수동 출석 체크 방식(호명, 서명)이나 카드/지문 인식 방식의 불편함, 부정 출석 위험, 위생 문제를 해결하고, 관리의 효율성과 정확도를 극대화하는 데 중점을 둡니다.

주요 목표 :

빠르고 정확한 실시간 출석 확인 및 데이터 관리

핵심 기술 :

얼굴 검출(Face Detection) 및 얼굴 인식(Face Recognition) 알고리즘

주요 대상 :

학교, 학원, 기업의 교육장, 회의실, 건설 현장 등 출석 관리가 필요한 모든 환경.

🎬 시연 영상

💻 화면 스크린샷

🧩 기술 구성

  • 백엔드: Spring Boot
  • 모바일: React Native (Expo)
  • DB: MySQL
  • AI: OpenCV, dlib
